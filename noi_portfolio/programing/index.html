■ jQueryのWebStorageを使用した初回のみのローディング処理

First-time-only loading process using jQuery's WebStorage

このポートフォリオサイトで使用しているローディング処理のソースコードになります。
jQueryのWebStorageを使用し、初回時にsessionStorageにデータをセット（setItem）。
2回目以降はデータを取得（getItem）し、セットされた値と一致すれば、
クラス「.start」を「display: none」しています。

This is the source code for the loading process used in this portfolio site.
Using jQuery's WebStorage, data is set in sessionStorage the first time (setItem),
data is retrieved the second and subsequent times (getItem), and if the set values match,
the ". start" class is "display: none".
