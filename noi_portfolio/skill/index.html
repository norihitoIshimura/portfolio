SKILL

自身のスキルを簡潔にまとめました。
フロントからバックエンドまで幅広く対応できるよう、PHPとJavaScriptを中心に勉強中です。
Here is a brief summary of my own skills.
I am studying mainly PHP and JavaScript in order to be able to handle a wide range of tasks from front to back-end.

HTML5 & CSS3

コーディング作業は、デザインカンプからのレスポンシブ対応など、HTML5とCSS3を使用した作業が可能です。エディターは「Visual Studio Code」または「Atom」を使用していますが、他のエディターも使用可能です。「Atom」については、開発が終了しているため、今後は「Zed」を使用する予定です。
Coding work can be done using HTML5 and CSS3, including responsive support from design comps. I use "Visual Studio Code" or "Atom" as my editor, but other editors can also be used.
As for "Atom," I will be using "Zed" from now on, since its development has been terminated.

PHP

PHPを用いてのWebシステム構築を勉強中です（データベースの管理に関しては、MySQLを利用しています）。
このポートフォリオサイトでは、PHPをベースにした動的なWebページやアプリのソースコード・サンプルを順次アップ出来ればと考えています。
I am learning to build web systems using PHP (as for database management, I use MySQL).
In this portfolio site, I hope to upload source code samples of dynamic web pages and applications based on PHP.

JavaScript

jQueryを用いた部分的に動作する基本的なコードを書くことができます。現在はJavaScriptを用いたフォームのバリデーションチェックやJavaScriptライブラリのReact.jsを勉強しています。
I can write basic code that works partially using jQuery. I am currently studying form validation checks using JavaScript and the JavaScript library React.js.

Illustrator

Photoshopと連携したデザイン制作を行えます。印刷物に関しては、商品デザイン・ロゴデザイン・パッケージデザイン・リーフレット等制作が可能です。Webデザインに関しては、デザインカンプ・バナー・アイコンやパーツの制作が可能です。
I can work with Photoshop to create designs. For printed materials, we can create product designs, logo designs, package designs, leaflets, etc. For web design, we can create design comps, banners, icons, and parts.

Photoshop

写真の補正・加工・レタッチ・合成・切り抜き作業が可能です。Webデザインにおいて、デザインカンプはIllustratorで作成していますが、Photoshopでのデザインカンプ作成も可能です。
I am capable of correcting, processing, retouching, compositing and cropping photos. Design comps are created in Illustrator, but we can also create design comps in Photoshop.

Indesign

カタログデータなどページ数の多いレイアウトデータにおいて、編集・操作を行う事ができます。
Editing and manipulation can be performed on layout data with a large number of pages, such as catalog data.
